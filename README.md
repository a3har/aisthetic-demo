# Aisthetic Demo

## Steps to run the project

1. Clone the repository
2. Ensure that mongodb is installed and running on your system
3. Run `npm install` in both `frontend/` and `backend/` dirs to install all the dependencies
4. Run `npm start` in `backend/` to start the backend server
5. Run `npm run dev` in `frontend/` to start the frontend server
6. Open `localhost:3000` in your browser to view the project

## For performance testing

1. Run `npm run build` in `frontend/` to build the frontend
2. Run `npm run start` in `frontend/` to start the server
3. Open `localhost:3000` in your browser to view the project
4. Run lighthouse audit in Chrome dev tools to check the performance of the website
